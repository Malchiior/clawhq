## SEO Technical Notes

- All blog posts should have proper H1/H2 structure
- Target 1 primary keyword + 3-5 secondary keywords per post
- Include internal links between blog posts
- Add schema markup (Article, FAQ, HowTo) where appropriate
- Meta descriptions: 150-160 chars with primary keyword
- URL slugs: keep short, include primary keyword
- Image alt text: descriptive, include keywords naturally
- Target featured snippets with clear, concise answer paragraphs
